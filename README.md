# FreshCart E-Commerce App

A frontend React application for an e-commerce platform, focusing on modern component design and state management. Built using Vite for fast development and optimized builds.

## Features
- Component Architecture: Structured React components for reusable UI elements.
- Context API: Manages global state (e.g., shopping cart data) across the application.
- Modern Build: Uses Vite for rapid hot module replacement (HMR) and optimized production builds.

## Tech Stack
- Framework: React.js
- Build Tool: Vite
- Language: JavaScript
- State Management: React Context API

## Current Status and Known Limitations
(Honest Assessment)
- Work in Progress: The application is currently a starting foundation and not a completed e-commerce solution.
- Missing Backend: No backend integration for actual product fetching or checkout processing.
- Default Content: Contains some default Vite boilerplate that needs to be cleaned up.

## How to Run Locally
1. Clone the repository
2. Install dependencies: npm install
3. Run the development server: npm run dev
